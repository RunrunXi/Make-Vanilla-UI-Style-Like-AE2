# 使原版UI的风格改为AE2的风格

这是一个适用于 Minecraft 的资源包，将原版UI风格改为 Applied Energistics 2 (AE2) 模组的风格。

## ✨ 主要内容

- **新材质**：将原版UI风格改为 AE2 模组风格的材质

## 🚀 安装方法

1. 下载最新的 `.zip` 文件
2. 将文件放入 Minecraft 的资源包文件夹：
   - Windows: `%appdata%\.minecraft\resourcepacks\`
3. 在游戏中选择选项 → 资源包 → 启用本材质包
